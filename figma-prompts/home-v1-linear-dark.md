# Home V1: Linear Dark Mobile

Design a premium mobile app Home screen for a habit tracker called MyApp. Create a single iPhone-sized screen, modern and polished, inspired by Linear, Notion Calendar, and high-end productivity apps. Dark theme only. Clean, sharp, minimal, not playful.

This screen is the main dashboard. It must support:
- top app bar with app name, notifications button, birthdays button
- primary CTA to add a habit
- a global month overview card showing all habits together
- a selected habit section with month navigation
- a habit picker/dropdown
- a habit-specific month calendar
- clear states for completed days, scheduled days, missed days, birthday markers, and note markers
- small hint that tapping a day marks it done and long-press opens a note
- bottom navigation with 5 tabs: Home, Who am I, Motivation, Tasks, Goals

Layout:
- top safe-area header with “MyApp”, compact bell icon, compact birthday/cake icon
- hero summary block under header with selected month, completion summary, and quick add button
- first card: “Month overview” for all habits, compact full-width calendar
- second card: selected habit area
- inside selected habit area include previous month button, current month label, next month button
- under that show a habit selector chip/dropdown with habit name and streak
- below that show the larger detailed month calendar
- floating add habit button or high-visibility pill CTA
- bottom nav anchored and refined, not chunky

Visual direction:
- black / graphite / charcoal base
- accent color: electric blue or teal
- success state: vivid green
- missed state: muted red
- birthday marker: small bright accent ring
- note marker: subtle dot
- use plenty of spacing, rounded 20-24 px cards, soft borders, low-noise shadows

Typography:
- use a clean sans serif such as SF Pro, Geist, or Inter Tight
- strong hierarchy with compact labels and medium-weight section titles
- avoid oversized text

Component notes:
- calendars should feel custom and premium, not default system widgets
- cells should clearly differentiate:
  - completed
  - scheduled
  - missed
  - disabled / outside interaction
- selected habit card should look more interactive than the global overview card

Output:
- one final mobile screen only
- include realistic sample content
- show at least one birthday marker and at least two note markers
- show a visible streak beside the selected habit name
- keep it elegant, minimal, and highly usable
