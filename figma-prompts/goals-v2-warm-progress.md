# Goals V2: Warm Progress Mobile

Create a modern mobile Goals screen for MyApp. The design should feel premium but approachable, with softer surfaces, clean spacing, and strong progress communication. Think productivity app meets personal growth journal.

The screen should include:
- top header with “Goals”
- clear add goal CTA
- two grouped sections:
  - Goals to achieve
  - Goals achieved
- expandable goal cards
- visible subgoals
- completion toggles
- rename, delete, add-subgoal, and expand controls

Design approach:
- light theme
- calm warm neutrals with a deep teal or forest accent
- success green for completed goals
- cards with soft rounded corners and generous internal spacing
- subtle progress visuals like completion counts or progress bars

Must communicate:
- active goals should feel motivating and actionable
- achieved goals should feel satisfying but visually quieter
- one active goal should be expanded to show subgoals
- if subgoals are incomplete, the parent done state should appear unavailable or secondary

Suggested layout:
- compact top bar
- under header, a summary strip like “4 goals in progress / 2 completed”
- first section: active goals
- second section: achieved goals
- each card:
  - goal title
  - compact progress indicator like 2/3 subgoals complete
  - main check icon on the left
  - utility actions on the right
- expanded state:
  - stacked subgoal rows
  - each row with toggle, text, and secondary actions

Typography:
- refined and readable mobile typography, such as Manrope, Plus Jakarta Sans, or SF Pro
- emphasize the active goal title and progress line

Sample content:
- active goal: “Run a half marathon” with 3 subgoals, 2 completed
- active goal: “Build portfolio site”
- achieved goal: “Read 12 books” with completion date

The result should feel optimistic, organized, and premium, while still being a realistic product UI.
