# Home V2: Soft Card Mobile

Design a mobile Home screen for a habit tracker app called MyApp. Create a modern premium iOS/Android hybrid interface with soft cards, calm colors, and very clear hierarchy. The style should feel like Headspace meets Stripe mobile, but for habit tracking. Still clean and minimal, but warmer and more human than a hard SaaS dashboard.

This screen needs to communicate:
- today’s focus
- this month across all habits
- one currently selected habit in detail
- fast access to notifications, birthdays, and add habit
- bottom navigation for Home, Who am I, Motivation, Tasks, Goals

Required content:
- top header with app name and 2 icon actions: notifications and birthdays
- summary strip showing current month, habits tracked, completion rate, and a primary add habit button
- global month overview card labeled “All habits”
- detailed selected habit card with:
  - month navigation arrows
  - habit selector
  - streak display
  - larger calendar
- compact onboarding hint for gestures: tap day = complete, long press = note

Design style:
- light theme with soft neutral background
- palette: warm white, stone, slate text, muted navy or deep teal accent, green success, coral miss/error
- large rounded cards with layered depth
- premium mobile spacing, breathable padding, no clutter
- gentle gradients or subtle tinted surfaces are okay

Calendar behavior to visualize:
- completed days = filled success state
- scheduled upcoming days = neutral tinted state
- missed days = soft coral state
- birthday markers = tiny celebratory accent
- note markers = small secondary dot

Layout guidance:
- mobile-first, single-screen composition
- header and summary feel editorial and calm
- first card can be denser and more overview-focused
- second card should become the clear focal point
- bottom nav should be slim, polished, and icon-led
- add habit action should feel obvious without overwhelming the screen

Typography:
- use a premium readable mobile font pairing, such as SF Pro + a softer display companion, or Manrope / Plus Jakarta Sans
- emphasize section titles and streak number

Use realistic sample data:
- selected habit: “Morning Run” with a 12-day streak
- one or two other habits implied in the all-habits view
- at least one birthday indicator and note indicators visible

The final result should feel premium, friendly, and highly practical for daily use.
