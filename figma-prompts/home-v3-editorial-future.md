# Home V3: Editorial Future Mobile

Create a bold mobile Home screen concept for MyApp, a habit tracking app. This is still a real product screen, not concept art, but it should feel more elevated and memorable than a standard productivity app. Think modern editorial UI mixed with elegant product design. Premium, minimal, and mobile-first.

Functional requirements:
- top header with app name and actions for notifications and birthdays
- strong monthly overview of all habits
- one selected habit with detailed monthly tracking
- add habit CTA
- bottom nav with Home, Who am I, Motivation, Tasks, Goals
- communicate tap-to-complete and long-press-to-note behavior in a subtle way

Screen structure:
- compact top header
- strong hero area with month title and a concise personal progress statement
- visually striking “All habits” month overview block
- selected habit module beneath it with:
  - previous / next month controls
  - habit selector with streak
  - large custom monthly calendar
- anchored floating add button or pill CTA
- elegant bottom nav

Style direction:
- dark premium theme
- palette: ink black, deep navy, ice gray text, acid lime or cyan accent, green completion, desaturated red for missed
- high contrast with restrained use of color
- mix sharp alignment with rounded surfaces
- subtle glass, glow, or gradient accents are allowed, but keep the UI usable and believable

Calendar rendering:
- must look custom and elevated
- completed days need a satisfying visual treatment
- missed days need a quieter but obvious treatment
- birthday marker and note marker must both exist
- inactive dates should fade back

Hierarchy:
- selected habit module should be the main focus
- all-habits card should feel analytical
- hero copy can be something like “You’re building consistency this month”

Typography:
- use an expressive but disciplined font direction such as Space Grotesk, Sora, or Geist
- keep labels small and clean
- use one standout metric for the streak

Sample content:
- month: September 2026
- selected habit: “Deep Work”
- streak: 18
- show a few completed days, a few missed days, and at least one upcoming scheduled day

Design one final high-fidelity mobile screen that looks like a launch-ready premium app.
