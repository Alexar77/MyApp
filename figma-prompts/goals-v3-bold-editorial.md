# Goals V3: Bold Editorial Mobile

Design a standout mobile Goals screen for MyApp that feels premium, modern, and slightly editorial while still being fully usable. This is for a real product, not just a visual experiment. Think strong hierarchy, dramatic simplicity, and refined mobile interactions.

Functionality that must be represented:
- screen title “Goals”
- add goal CTA
- collapsible groups for:
  - Goals to achieve
  - Goals achieved
- goal cards with expand/collapse
- subgoal rows inside expanded cards
- actions for add subgoal, rename, delete
- done toggles for goals and subgoals

Important behavior to convey:
- a goal can only be completed when all subgoals are completed
- show this rule in the UI with a subtle helper message, disabled main toggle, or locked visual state on incomplete goals

Visual style:
- dark premium mode
- palette: black, deep slate, cool gray, vivid lime or cyan accent, green completion, muted rose for destructive actions
- strong contrast and disciplined spacing
- use one expressive typeface such as Space Grotesk or Sora for headers, paired with a clean UI sans for body
- cards can feel slightly architectural, with bolder shapes and very deliberate alignment

Composition:
- compact top header
- large elegant section headers with counts
- active goals first and visually dominant
- achieved goals lower and quieter
- one expanded active goal card showing 4 subgoals
- one achieved goal card showing completion date
- floating or anchored add goal CTA should feel premium and obvious

Card details:
- left side: status icon
- center: goal title and supporting line
- right side: add subgoal, rename, delete, expand icons
- expanded content should feel clean and nested without becoming crowded

Sample content:
- active goal: “Launch my freelance brand” with 4 subgoals, 3 complete
- achieved goal: “Save emergency fund” completed on 14 Sep 2026

Output one high-fidelity mobile screen with a premium product feel and a memorable visual identity.
